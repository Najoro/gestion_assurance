## Run Locally

Clone the project

```bash
  git clone https://github.com/Najoro/gestion_assurance.git
```

Go to the project directory

```bash
  cd gestion_assurance
```

Install dependencies

```bash
  composer install
  yarn install
  yarn dev or yarn watch
```

Start the server

```bash
  symfony server
```
